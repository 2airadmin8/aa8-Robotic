# 公開前チェックリスト

## 内容確認

- [ ] 製品名、メーカー名、型番を公式資料と照合
- [ ] 価格、納期、在庫を固定値として断定していない
- [ ] SDK・ROS・制御範囲を過大表現していない
- [ ] 保証、修理、規制、物流の未確認事項を明示
- [ ] 進行中案件を完了事例として表現していない
- [ ] 顧客名、ロゴ、写真、コメントの公開許可を確認
- [ ] 個人情報、非公開価格、大学内部資料を含めていない

## ページ確認

- [ ] Title、Description、H1の目的が一致
- [ ] 主キーワードが他ページと重複していない
- [ ] Canonicalが公開URLと一致
- [ ] Sitemapへ追加済み
- [ ] 製品・用途・支援・資料・相談の内部リンクがある
- [ ] 戻るリンクとパンくずが正しい
- [ ] 404リンクや空リンクがない

## モバイル確認

- [ ] iPhone Safariで横スクロールがない
- [ ] メニューを開閉できる
- [ ] Dialogを閉じられる
- [ ] 比較表が画面外へはみ出さない
- [ ] 長い製品名、URL、メールアドレスが崩れない
- [ ] ボタンが押しやすく、重なっていない
- [ ] FooterがDialog内部へ誤挿入されていない

## 機能確認

- [ ] 製品フィルターが動作
- [ ] 2〜3製品の比較が動作
- [ ] 比較URLコピーが動作
- [ ] 印刷・PDF保存が動作
- [ ] チェックリストの保存・コピー・消去が動作
- [ ] 問い合わせの下書き保存・復元が動作
- [ ] URLパラメータから製品・用途・事例を引き継ぐ
- [ ] メールアプリ起動と本文コピーが動作

## 計測・検索確認

- [ ] GA4イベントに個人情報を含めていない
- [ ] 比較、PDF、チェックリスト、問い合わせイベントを確認
- [ ] robots.txtがクロールを許可
- [ ] Sitemap URLが正しい
- [ ] Search Console優先ページを確認
- [ ] GitHub Actionsがすべて成功

## 公開後確認

- [ ] 公開URLで最新内容を確認
- [ ] CSS・JavaScriptの旧キャッシュが残っていない
- [ ] トップ、製品、支援、事例、問い合わせを通しで確認
- [ ] 重大不具合があれば直前の正常コミットを特定
- [ ] 修正コミットのActions成功後に完了報告
